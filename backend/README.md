# GasLink Rwanda - Backend API

A production-grade backend for the GasLink Rwanda LPG distribution system.

## Quick Start

### Prerequisites
- Node.js 18+
- MongoDB 7+
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies
3. Set `MONGODB_URI` in `.env`
4. Start the development server with `npm run dev`
5. Seed demo users with `npm run db:seed`
